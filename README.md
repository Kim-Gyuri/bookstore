# :pushpin: Improved-SpringBoot-Online-Shopping-Stor
## Features
+ MVC  - 로그인 처리(based 쿠키, 세션, 인터셉터), 파일 업로드(상품 이미지 등록)
+ Querydsl - 페이징처리( 모든상품 정렬/ 카테고리 정렬/ 상품명 검색)
+ CRUD
  + 회원등록, 회원수정, 회원조회
  + 상품등록
  + 상품조회 ( 회원ID로 조회, 카테고리별 조회,  상품 검색)
  + 로그인, 로그아웃 
  + 상품구매, 장바구니
  
## Technology Stacks
` Backend`
+ Java 11
+ Spring Boot 2.7.1
+ Spring Data JPA
+ Lombok
+ Hibernate
+ Spring Web
+ validation
+ H2 Database 
+ Querydsl
+ Gradle
